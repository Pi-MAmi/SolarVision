# Contributing to SolarVision

Thank you for your interest in contributing to SolarVision!

## How to Contribute

1. Fork the repository.
2. Create a feature branch.

```bash
git checkout -b feature/my-feature
```

3. Make your changes.
4. Test your changes on Raspberry Pi.
5. Commit using clear commit messages.

Example:

```bash
git commit -m "Add MQTT support"
```

6. Push your branch.

```bash
git push origin feature/my-feature
```

7. Open a Pull Request.

---

## Coding Guidelines

- Keep code readable.
- Follow existing project structure.
- Add comments where necessary.
- Test before submitting.

---

## Reporting Bugs

Please include:

- Raspberry Pi model
- OS version
- Python version
- Inverter model
- Error logs
- Steps to reproduce

---

## Feature Requests

Feature requests are welcome.

Planned roadmap includes:

- Multi Inverter
- MQTT
- Docker
- PostgreSQL
- Home Assistant
- Plugin System
